GET /subscribers
Get all subscribers with filtering options
GET /api/subscribers?status=active&premium=true&limit=100 Response: { "success": true, "data": [ { "id": 1, "email": "user@example.com", "name": "John Doe", "status": "active", "premium_status": false, "games_played": 15, "created_at": "2024-01-15T10:30:00Z" } ], "pagination": { "page": 1, "per_page": 100, "total": 1247 } }
POST /subscribers

üìß Email Sending

POST /email/send
Send individual email using template
POST /api/email/send Body: { "to_email": "user@example.com", "template_name": "welcome_email.html", "subject": "Welcome to Daily Puzzle Post!", "variables": { "name": "John Doe", "signup_date": "January 15, 2024" } } Response: { "success": true, "message": "Email sent successfully", "email_id": "email_12345" }
POST /email/bulk
Send bulk emails to subscriber segments
POST /api/email/bulk Body: { "template_name": "daily_challenge.html", "subject": "Today's Word Challenge", "target_segment": { "status": "active", "premium_status": false }, "variables": { "challenge_word": "ELEPHANT", "category": "Animals", "hint_text": "Large mammal with a trunk" } } Response: { "success": true, "campaign_id": 24, "emails_queued": 1089, "estimated_delivery": "2024-01-15T11:00:00Z" }
